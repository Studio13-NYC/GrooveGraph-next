# Screenshots

Use for **manual** captures or as an optional output directory for **Playwright** / other runners (configure `outputDir` or artifact upload to point here).

Image and video files under this directory are **gitignored** by default so the repo does not fill with binaries. Commit only when you intentionally add a small, reviewed asset (then use `git add -f` or adjust `.gitignore` for that file).

Suggested naming: `<date>-<feature>-<step>.png` or `e2e-<spec>-<viewport>.png`.
