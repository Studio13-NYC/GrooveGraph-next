# Visual System Source

This directory contains operational visual-system primitives for GrooveGraph Next.

## Current contents

- `tokens.ts`: palette, typography, and geometry tokens for both visual regimes

## Intended growth

Over time this directory can hold:

- prompt-building helpers
- SVG template generators
- route-map layout helpers
- title-card utilities

The purpose is to give the `Graphic Artist` lane repeatable substrate, not just conceptual style guidance.
