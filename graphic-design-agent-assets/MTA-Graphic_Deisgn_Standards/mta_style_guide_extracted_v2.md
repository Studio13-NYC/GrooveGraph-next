## Page 004 — Introduction
**Image:** [NYCTA GS Manual_text_Page_004.jpg](./NYCTA%20GS%20Manual_text_Page_004.jpg)

Key text, cleaned:

The graphics standards in this manual are the result of a design program intended to create a coordinated and effective rapid-transit signage system.

Standards for identification, directional and informational signage, maps, and color are designed to orient people within the subway environment.

The basic rule is that the rider should be given information only at the point of decision:

- Never before
- Never after

Three basic categories of signs are defined:

1. **Identification** — signs naming the station
2. **Directional** — signs leading the passenger from street to station to train, or the reverse
3. **Information** — signs about schedules, tracks, and facilities not directly related to transportation

The system is intended to create visual consistency, reduce clutter, and eliminate misleading or repetitive information.

## Page 013 — Letter Spacing System
**Image:** [NYCTA GS Manual_text_Page_013.jpg](./NYCTA%20GS%20Manual_text_Page_013.jpg)

This page explains the modular spacing system for the standard typeface.

Key guidance visible:

- A modular system is used to ensure consistent spacing for letters and words
- The unit system must be followed exactly
- The chart helps determine correct spacing for the “Standard Medium” typeface
- To find the spacing for a word, select the following letter from the top row and the preceding letter from the left column
- Word spacing is set to **one-half the height of the capital letter**
- If environmental conditions force size reduction, reduction should be photographic and proportionally controlled
- Arbitrary custom “add sizes” should be avoided

The page includes a detailed spacing matrix for uppercase and lowercase combinations.

## Page 014 — Color Coding
**Image:** [NYCTA GS Manual_text_Page_014.jpg](./NYCTA%20GS%20Manual_text_Page_014.jpg)

This page defines route color coding.

Key guidance visible:

- Each subway line is assigned a color code with associated letters or numbers
- These colors are used on directional signage and on train/line identification discs
- Discs are arranged in **alphabetical order first**, then **numerical order**
- Reproduction masters for discs are provided in the manual and should be the sole source for reproduction
- When reduction is required, it should be done photographically rather than redrawn manually
- Suppliers should match the provided swatches for color standards

Disc sizes referenced on the page include:

- **11 in. discs**
- **5.5 in. discs**
- **1.25 in. discs**

# Consolidated Design Rules Inferred from These Pages

## Information Architecture
- Show information only at the point of decision.
- Separate signage into identification, directional, and informational categories.
- Reduce clutter and repetition.

## Typography
- Use one consistent sans-serif type system throughout.
- Design around X-height rather than nominal point size alone.
- Keep directional signs brief; informational signs can take more lines but remain structured.
- Maintain strict spacing and leading rules.
- Use a modular grid system.

## Symbols and Route Bullets
- Use filled circles as the base shape for route bullets.
- Set letters/numbers in high-contrast white on dark fields.
- Center content optically, not just mechanically.
- Scale double-letter forms to preserve balance within the circle.

## Color
- Route colors are standardized and should be reproduced from approved masters.
- Colors should be matched to swatches, not approximated by eye.
- Disc order follows alphabetical sequence, then numerical sequence where applicable.

## Page 064 — Sign Plate Modulation
**Image:** [NYCTA GS Manual_text_Page_064.jpg](./NYCTA%20GS%20Manual_text_Page_064.jpg)

This page explains the modular sign-plate system.

Key rules, cleaned from the page:

- A modular system was developed to avoid visual clutter from signs of unrelated sizes
- Standard plate dimensions are based on **1, 2, 4, and 8 foot** widths by **1 foot** height
- The system standardizes both production and maintenance
- The recurring module improves visual recognition of sign messages
- Structural details should accommodate the modular sign concept rather than force ad hoc deviations
- A **1.5 inch black band** at the top of the panel is part of the sign-plate system

Examples on the page indicate typical plate sizes by category:

- **1 × 1** — line-identification numbers, informational letters, arrows, and mandatory messages
- **2 × 1** — smaller informational units
- **4 × 1** — directional signage
- **8 × 1** — station-identification signage

## Page 067 — Exit and Transfer Signs
**Image:** [NYCTA GS Manual_text_Page_067.jpg](./NYCTA%20GS%20Manual_text_Page_067.jpg)

This page defines the treatment of exit and transfer signage.

Key rules, cleaned from the page:

- These signs must be especially noticeable
- **Exit** signs are **white on red**
- **Transfer** signs are **black on yellow**
- The larger importance of these signs justifies larger, high-visibility type
- Exit signs use a **2 × 1** module
- Transfer signs use a **4 × 1** module
- When arrows and route discs are included, the **arrow precedes** the wording and the **route discs follow**
- When both transfer and exit information appear on one sign, the directional logic must remain unambiguous
- Transfer information on platforms should avoid being confused with straight-ahead exit guidance

Visible examples include:

- **Exit**
- **Transfer**
- **1 2 3** route discs paired with transfer text

## Page 087 — Temporary Signs
**Image:** [NYCTA GS Manual_text_Page_087.jpg](./NYCTA%20GS%20Manual_text_Page_087.jpg)

This page describes how temporary signs should fit into the system.

Key rules, cleaned from the page:

- Temporary signs should still use the standard visual system
- Type size should be consistent
- Typeface should remain **Standard Medium**
- The same modular widths apply: **1, 2, 4, 6, or 8 feet**
- Standard spacing and leading rules still apply

Production guidance visible on the page:

- Use pre-aligned, pre-spaced self-adhesive vinyl letters for small quantities
- For larger runs of the same temporary message, use **silkscreen**
- The temporary message should be **black on white**
- Transparent material may be used with white lettering on dark surfaces and black lettering on lighter ones
