Typography

Baseline: Helvetica stack — Helvetica Neue, Helvetica, Arial, Nimbus Sans L. Hierarchy follows NYCTA signage roles: identification, directional, informational. Type scale tokens: --gg-type-grid-1 (informational), --gg-type-grid-2 (directional), --gg-type-grid-3 (identification).

Non-negotiable: no decorative display faces for primary system UI. Use weight, spacing, and alignment for hierarchy.

Identification

Station · system · object name

Largest stable title; names the place or artifact.

Directional

Context / Active corridor

Uppercase, letterspaced; orange marks the current path.

Informational

Metadata, timestamps, provenance — smaller, slate-tinted, bounded line length. Directional signs: ≤ ~2 lines at scale; informational modules may run longer but stay modular.

Lines & curves

Uniform trunk weight, parallel gutters (~0.52× stroke), engineered radii — no organic Bézier drift for route logic. Map construction uses 0°, 90°, and 45° segments. Bend corners use a five-step radius family (~×√2 between steps).
