New-regime interactive style guide

NYCTA-inspired civic information design: typography in three layers, route colorways as durable meaning, engineered stroke and radius grammar, and vector transit logic you can probe. Canonical written spec: docs/VISUAL_STYLE_GUIDE.md in the repository root.

Typography Colorways Strokes Radius Vector map Motion Regimes

Typography

Studio13-style surfaces default to a neutral transit sans. Separate identification (what this is), directional (where you are and what happens next), and informational (metadata and support) before adding ornament.

Identification

Governance map · routing chart

Very large bold title; object names read as signs, not marketing headlines.

Directional

Framework / Slice / Smoke test

Actions, breadcrumbs, and section markers use weight and letterspace; orange marks the active corridor (directional accent).

Informational

Metadata, timestamps, provenance, and secondary copy stay smaller and slate-tinted so rails and titles remain the primary signal. Minimal body copy; let the line and label hierarchy carry the story.

Colorways

Route colors name families or durable system meaning, not mood. Paper and ink ground the map; slate supports secondary labels. Click a swatch to copy its hex.

Tip: canonical tokens: framework/src/visual-system/nycta-groovegraph-tokens.css — import @groovegraph-next/framework/nycta-groovegraph-tokens.css. --gg-map-land mirrors --gg-surface-canvas (neutral; no beige).

Stroke scale

Hairline for guides, 2× for secondary rails, 3× for primary trunks and dominant graph routes.

1× hairline 2× route 3× trunk

Radius family

Corners and bends use a repeatable radius family so the system feels engineered, not hand-drawn: R1 tight, R2 standard, R3 broad sweep.

R1 — compact elbow

R2 — standard bend

R3 — major sweep

Vector grammar

Hover routes to isolate families; bundled parallels share a corridor then diverge; interchange reads as compression. Toggle labels to mimic progressive disclosure.

Probe: hover a route to read its role.

Motion stance

Motion is directional and structure-bearing: strokes draw on, nodes snap, staggers read at a glance. Prefer transform and opacity; honor prefers-reduced-motion (this page scales animations down when set).

Implementation lane: gsap + @gsap/react with explicit plugin registration — see .cursor/skills/gsap-motion-system/SKILL.md.

Regime contrast

Authoritative artifacts use new-regime language. Retrospective satire switches to whiteboard energy — only blend when contrast is the point.
